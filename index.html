What is this?

Every wanted to have your Dribbble shots automagically on a portfolio website with your custom domain? You've found the right project: Basketball. It's a self-hosted alternative to the paid Dribbble Playbook. All you need for this to work is:

- Server (min. 1GB RAM / 5GB HDD/SSD) or Serverless enviroment (e.g. Vercel/DETA)
- Redis Database (e.g. Upstack)
- preferably a domain

RECOMMENDED: Cloudflare SSL

The server or serverless enviroment you use has to support FastAPI and have Python 3.6+ suppport. Check if your server supports Python with python -v or python3 -v. To check if your serverless enviroment supports Python, consolidate your provider's website or documentation.
